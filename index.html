Financial Modeling Prep · fallback opcional Sin configurar

La fuente principal por ticker es Yahoo Finance (gratuita, sin API key). FMP queda como fallback opcional para quien tenga plan Starter+ ($14/mes): si Yahoo no devuelve datos para un ticker, se reintenta automáticamente con FMP si la API key está configurada aquí.

⚠ El plan Basic gratuito de FMP dejó de cubrir los estados financieros históricos en 2024-2025. Con plan Basic verás errores 402 (Payment Required). Solo configura FMP si tienes Starter+.
Si decides activar FMP como fallback:
  1. Ve a financialmodelingprep.com/developer/docs
  2. Suscríbete a un plan que incluya los endpoints financial-statements (Starter o superior)
  3. Dashboard → copia la API key
  4. Pégala arriba — se guarda en este navegador (no la enviamos a ningún sitio)

Sincronización GitHub Sin configurar

Sincroniza watchlist y análisis entre dispositivos (móvil, portátil, etc.) usando un repositorio privado tuyo en GitHub. Sin servidores que mantener, sin pagar nada.
Setup en 5 minutos:
  1. Crear repo privado vacío en github.com/new (ej: sabi-analyzer-data) — marca Private
  2. Generar Personal Access Token en github.com/settings/tokens (fine-grained):
    • Repository access: Only select repositories → tu repo nuevo
    • Permissions: Repository permissions → Contents: Read and write
    • Expiration: 1 año (o "no expiration")
  3. Copiar el token (empieza por github_pat_) y pegarlo arriba
  4. Owner/repo: tu usuario/nombre-del-repo (ej: miquel/sabi-data)
  5. Pulsa Probar conexión para verificar
⚠ Importante: el token vive solo en este navegador (localStorage). Si lo borras o cambias de dispositivo, vuelve a configurarlo. Nunca lo subimos a ningún sitio externo.
Privacidad: todas las claves y tokens se guardan únicamente en este navegador (localStorage). No se envían a ningún servidor externo nuestro. Las llamadas a FMP van directamente desde tu navegador a su API. La sync de GitHub va de tu navegador a tu propio repo privado.
